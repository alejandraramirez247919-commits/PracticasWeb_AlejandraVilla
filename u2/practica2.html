¿Que es Internet?

Internet es una red global de computadoras interconectadas que utiliza un conjunto de protocolos estándar, principalmente el Protocolo de Internet (IP), para permitir el intercambio universal de datos e información. Funciona como la infraestructura de comunicación mundial.

¿Tipos de Proveedore de internet y nombres?

Un Proveedor de Servicios de Internet (ISP) es la empresa que ofrece a los usuarios la conexión a la red. Los tipos se clasifican principalmente por la tecnología de conexión que utilizan: el ISP de Fibra Óptica (como Google Fiber o Movistar) ofrece velocidades ultrarrápidas a través de cables de vidrio; el ISP de Cable (como Xfinity o Izzi) usa la infraestructura de televisión por cable; el ISP de DSL (como AT&T) utiliza líneas telefónicas de cobre; el ISP Móvil (como T-Mobile o Telcel) ofrece acceso a través de redes celulares 4G/5G; y el ISP Satelital (como Starlink o Viasat) proporciona conectividad en zonas remotas mediante satélites en órbita

¿Que es www?

WWW significa World Wide Web y es un sistema de documentos de hipertexto interconectados (páginas web) que se accede a través de Internet. Es el servicio más popular que funciona sobre la infraestructura de Internet, permitiendo a los usuarios navegar mediante enlaces.

¿Que es carpeta local y carpeta remota?

Una carpeta local es un directorio de archivos que reside en el dispositivo de almacenamiento físico de tu propia computadora o dispositivo. En contraste, una carpeta remota es un directorio de archivos alojado en un servidor o computadora diferente a la tuya, al cual se accede a través de una conexión de red.

¿Que es FTP?

FTP (File Transfer Protocol) es un protocolo de red estándar utilizado para transferir archivos y datos entre un cliente (tu dispositivo) y un servidor a través de una red, siendo el método principal para subir y descargar archivos grandes, como páginas web completas.

¿Que es experiencia de usuario?

La Experiencia de Usuario (UX) es la totalidad de sentimientos, emociones y percepciones que experimenta una persona al interactuar con un producto digital o servicio. Se enfoca en que la interacción sea efectiva, eficiente, placentera y satisfaga las necesidades del usuario.

¿Que es interfaz de usuario?

La Interfaz de Usuario (UI) es el punto visual y tangible de interacción con un sistema o software, abarcando todos los elementos gráficos (botones, tipografía, imágenes, campos) con los que el usuario interactúa para controlar y usar la aplicación.

¿que es un DNS?

El DNS (Domain Name System) es el sistema de traducción de Internet que convierte los nombres de dominio legibles para humanos (ej. https://www.google.com/search?q=google.com) en las direcciones IP numéricas (ej. 172.217.10.46) que las máquinas utilizan para encontrar y comunicarse entre sí en la red.

¿Que es un Hosting?

El Hosting o Alojamiento Web es el servicio que consiste en alquilar un espacio en un servidor conectado a Internet para almacenar todos los archivos y datos necesarios de un sitio web, garantizando que este esté disponible para cualquier persona en el mundo 24 horas al día.

¿Que es un full stack designer?

Un Full Stack Designer es un profesional del diseño con un amplio conjunto de habilidades que abarca tanto el diseño de la Experiencia de Usuario (UX) y la Interfaz de Usuario (UI), como un conocimiento práctico de la codificación (frontend y, a veces, backend) necesaria para implementar y dar vida a sus diseños, permitiéndole supervisar el producto desde la concepción hasta el lanzamiento.
